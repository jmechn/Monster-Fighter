IMPORTING TO ECLIPSE
1. Extract the .zip and save it into your Eclipse workspace.
2. Make sure the name is 'jch423_ayu29_MonsterFighter'
3. Open Eclipse
4. Navigate to File > Import then General > Existing Projects and choose the folder and press finish.
5. Press Run > Run > Java Application > TitleScreen > Okay

RUNNING THE JAR FILE
1. Extract .zip to a known destination
2. Change directory to the destination in terminal
3. Type 'java -jar jch423_ayu29_MonsterFighter.jar'
